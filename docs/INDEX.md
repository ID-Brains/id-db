# Welcome to the project-hikma

Welcome to the **project-hikma** (idbrains-db project)! This is a community-driven, open-source project dedicated to making academic knowledge accessible and collaborative for all students.

Whether you're here to learn, contribute, or maintain, this documentation will guide you through our mission and how to get involved.

**Note that** : i mention this project by `project-hikma` because we haven't decided a name yet but the codename of this project internally is `Hikma`.

- **[Contributing Guide](./CONTRIBUTING.md):** Ready to share your notes or expertise? Start here!
- **[Content Guidelines](./CONTENT_GUIDELINES.md):** Detailed instructions on how to format your content.

## 🤝 Community & Support
This project thrives on student participation. If you find a bug, have a suggestion, or want to contribute a set of lecture notes, please join the conversation on [GitHub Discussions](https://github.com/ID-Brains/id-db/discussions) or Giscus and we will provide a link to a discord server later.
