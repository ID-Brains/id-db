# Roadmap & Open Questions

This document is a living draft of our future plans for the **College Knowledge Base**.

## 📍 Project Roadmap

### Phase 1: Core Content & Automation
- [ ] Finalize the [Content Guidelines](./CONTENT_GUIDELINES.md) and frontmatter schema.
- [ ] Implement the automated [Validation Pipeline](./ARCHITECTURE.md#1-validation-validateyml).

### Phase 2: Community & Growth
- [ ] Recruits [Subject Leads](./GOVERNANCE.md#subject-leads) for key subjects.
- [ ] Launch with a core set of notes and summaries for Year 2.
- [ ] Enhance the [Quiz System](./CONTENT_GUIDELINES.md#3-quizzes-and-exams) with better feedback and explanations.

### Phase 3: Long-term Sustainability
- [ ] Explore [OCR Fallback](./ARCHITECTURE.md#2-text-extraction-extractyml) for scanned PDFs (Optional).
- [ ] Integrate [Google Drive API] for large file hosting.
- [ ] Launch the [Web Form](./CONTRIBUTING.md#1-the-web-form-easiest) for simplified contributions.
- [ ] Build a search-centric landing page for easier discovery.
